# EasyNode

VPS 一键节点部署工具

## 复制下面的命令，粘贴到服务器后，回车即可。

```bash
curl -fsSL https://raw.githubusercontent.com/pangzibaike/easynode/main/easynode.sh | bash
```

## 特性：

- 首次使用：运行脚本即可完成部署并生成节点。
- 如果 VPS 重启，需要再次运行脚本，以获取最新的 Cloudflare Quick Tunnel 地址。
- 如果想更换节点链接，也可以再次运行脚本，脚本会生成新的 Quick Tunnel，并更新 node.txt。
